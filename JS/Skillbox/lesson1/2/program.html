Описание Курса

В первой частьи курса ву познакомились сц стандартами Web 2.0 научитесь работать с макетами и форматорпвать код,освоите адаптивную верстку,а по итогу-сможете создавать быстрые и удобные сайты,которые точно понравятся пользователям.Вы поймете назначение Основныхй тегов HTML.сможетебез усилийстилизировать и размещать элементы с помощю CSS и сверстайте свой первый макет.

Вторая часть курса сделает из вас настоящего мастера.Вы изучите сборщики проектов,научитесь верстать email-рассылки,разберетесь с анимацей на сайте.

Помимо этого ,вы сверстайте фрагмент из интернет-магазина - главнаю страницу ,страницу каталога и страницу товара!
